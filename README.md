# 🛍️ Customer Shopping Behavior Analysis
Overview

This project analyzes Customer Shopping Behavior to uncover valuable insights into customer purchasing patterns, product preferences, payment methods, and sales performance. The project follows a complete data analytics workflow—from importing and cleaning raw data to building an interactive Power BI dashboard and presenting business recommendations.
Using Python for data preprocessing and exploratory data analysis (EDA), SQL (PostgreSQL, MySQL, and SQL Server) for business queries, and Power BI for visualization, the project demonstrates how raw customer data can be transformed into actionable business insights.
